# CommandAuction - 指令拍卖插件

一个Minecraft Spigot 1.12.2插件，允许以OP身份执行指令权限的拍卖系统，拥有美观的GUI界面。

## 功能特点

- 通过控制台或有权限的玩家启动指令执行权限拍卖
- 支持设置起拍价、一口价和拍卖持续时间
- 美观的GUI界面，方便玩家查看拍卖信息和参与竞拍
- 支持自动10%加价和自定义出价
- 兼容PlayerPoints和Vault经济系统
- 以OP身份安全执行拍卖的指令

## 安装说明

1. 确保服务器已安装以下依赖插件：
   - Vault
   - PlayerPoints (可选)
   - 一个与Vault兼容的经济插件（如Essentials）

2. 将`CommandAuction.jar`放入服务器的`plugins`文件夹

3. 重启服务器

## 命令说明

- `/auc start <起拍价> <一口价> <持续时间(秒)> <执行命令>` - 开始一个新的拍卖（需要管理员权限）
- `/auc bid <金额>` - 对当前拍卖进行出价
- `/auc buy` - 以一口价购买当前拍卖
- `/auc gui` - 打开拍卖GUI界面
- `/auc end` - 手动结束当前拍卖（需要管理员权限）
- `/auc` - 查看帮助信息

## 权限设置

- `commandauction.admin` - 允许使用管理员功能（默认为OP）
- `commandauction.use` - 允许参与拍卖（默认为所有玩家）

## 使用示例

1. 启动一个给予飞行权限的拍卖：
   ```
   /auc start 100 500 300 fly
   ```
   这将创建一个起拍价100，一口价500，持续5分钟的拍卖，获胜者将获得执行`/fly`指令的权限。

2. 玩家通过GUI参与竞拍或使用命令出价：
   ```
   /auc bid 150
   ```

3. 玩家可以选择一口价直接购买：
   ```
   /auc buy
   ```

## 注意事项

- 请谨慎设置拍卖的指令，确保不会被滥用
- 拍卖的指令将以临时OP权限执行，请确保服务器安全
- 插件会自动处理资金交易，包括退还失败的出价

## 兼容性

- Minecraft Spigot 1.12.2
- Vault 1.7.x
- PlayerPoints 3.0.x+
